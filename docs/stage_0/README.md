Hardware Features
=================
 - Xilinx Kirtex-7 XC7K325T FFG900 -3
 - Four SFP+ interface (using 4 GTX transeivers)
 - Supports both 10Gps and 1Gps modes
 - x8 PCI Express Gen3
 - Two x36 QDR II (CY7C1515KV18)
 - DDR3 SODIMM

DDR3
=====
  - Bank 32: DQ[32-63]
  - Bank 33: Address/ctrl
  - Bank 34: DQ[0-31]

QDR II 0 
========
   - Bank 16: D[0-35]
   - Bank 17: Address/ctrl
   - Bank 18: Q[0-35]

QDR II 1 
========
    - Bank 12: D[0-35]
    - Bank 13: Address/ctrl
    - Bank 14: Q[0-35]
